## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code recommended)
- Optional: Live Server extension

### Installation

#### 1️⃣ Clone the Repository
```bash
git clone https://github.com/MinalKanth/biome-logistics.git
cd biome-logistics
```

#### 2️⃣ Open in Browser
```bash
# Method 1: Direct open
# Double-click index.php

# Method 2: With Live Server
# Right-click index.php → "Open with Live Server"
```

#### 3️⃣ Customization

**Update Company Information:**
```html
<!-- In all files, search & replace: -->
- "Your Company Name" → "Biome Enterprises"
- "+91-XXXX-XXXX-XX" → Your actual phone
- "example@email.com" → Your email
- "Your Location" → Your office address
```

**Change Colors:**
```css
/* In CSS :root variables */
--primary: #0F4C2D;      /* Green */
--accent-green: #8BC34A; /* Bright Green */
--gold: #F5A800;         /* Gold */
```

**Add Your Logo:**
```html
<!-- In navbar -->
<img src="your-logo.png" alt="Logo" height="50">
```

```bash

2. Click "New Project"
3. Import GitHub repository
4. Click "Deploy"
```

```bash

3. Select "main" branch as source
```
